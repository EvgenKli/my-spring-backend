# Cloud-Native Java Spring Boot REST API Backend

Промышленное бэкенд-приложение на базе фреймворка Spring Boot, функционирующее в распределенном облачном кластере.

## Функционал приложения
Реализован REST API для управления сущностями пользователей:
* `POST /users` — Прием данных в формате JSON и запись в удаленную облачную СУБД PostgreSQL.
* `GET /users` — Чтение и структурирование данных из базы (ORM Hibernate / Spring Data JPA).

## Спецификация контейнеризации
Разработан оптимизированный **Multi-stage Dockerfile** на базе легковесного дистрибутива `eclipse-temurin:21-jre-alpine`. Вес финального продакшен-образа сокращен с 500 МБ до 50 МБ. Образ автоматически доставляется в Docker Hub Registry.
